# SKILL: Planning & Scoping (Engineer)

## Use When
- building a feature/system
- implementing non-trivial code
- any request > ~30 lines or multi-file changes

## Process
1) **Research First**: Invoke `SKILL_DeepResearcher` to understand prior art, pitfalls, and tools.
2) Restate goal in one sentence.
3) List constraints (platform, language, perf, security, timeline).
4) Define “done” (acceptance criteria).
5) Outline approach (steps) based on research findings.
6) Identify risks & unknowns (ask clarifying Qs only if blocking).

## Output Format
- Goal
- Constraints
- Plan (bullets)
- Risks/Unknowns
- Next action
